📖 项目简介 / Overview
这是一个专为招投标标书和高标准工程排版设计的网络图生成工具。不同于市面上常见的流程图工具，本项目核心采用深度优化的集合论推演算法，能够自动处理复杂的工序逻辑、平行作业及流水施工交叉，生成符合中国工业标准的横平竖直双代号网络图。

Designed for high-specification bidding documents and engineering layout. Unlike generic flowchart tools, this engine utilizes optimized Set Theory algorithms to automatically handle complex sequence logic and parallel tasks, generating standard orthogonal double-coded network diagrams.
